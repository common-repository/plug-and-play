== Description ==
<strong>Plug and Play</strong> our feautures and turn your WordPress Blog into a <strong>Highly Interactive, Elegant and Secure</strong> Blog.

[Read more](https://wordpress.org/plugins/plug-and-play/) about with this plugin!

<h4>Features</h4> 
* Hide Admin Bar:
	Disable Wordpress Admin bar for front end user, Speedup your web site loading.
* WP Generator Meta Tag Remover:
	Customize your own website loogin form logo.
* Security Tools:
		It can be considered a security risk to make your wordpress version visible and public you should hide it.
* Versus:
	Shows recently viewed posts by visitor as a sidebar.
* Maintenance Mode:
	Easily create a maintenance mode page for your WordPress site.

== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for Plug & Play
* Click to install
* Activate the plugin 
* Plug and Play our feautures for WordPress to Turn your WordPress Blog into a Highly Interactive Blog.
